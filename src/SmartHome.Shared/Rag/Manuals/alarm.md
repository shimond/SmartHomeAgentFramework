# Aria Home Alarm Panel

## Resetting the master code
Enter the current master code, press MENU, then 9. Enter the new 4-6 digit code,
press OK, then re-enter to confirm. If the code is lost, hold RESET for 15 seconds
to enter installer mode; this also disarms the system.

## Arming modes
HOME (perimeter only) and AWAY (perimeter and motion).
